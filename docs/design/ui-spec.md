## 1. Дизайн-система

### 1.1 Цветовая палитра

#### Статусные цвета (семантические)

| Статус | Название | Light HEX | Dark HEX | Применение |
|--------|----------|-----------|----------|------------|
| OK | Green | `#34C759` | `#30D158` | Потребление 0-69% лимита |
| Warning | Yellow | `#FF9F0A` | `#FFD60A` | Потребление 70-89% лимита |
| Critical | Red | `#FF3B30` | `#FF453A` | Потребление 90-100% лимита |

#### Пороги переключения статуса

```
0%  ──────────── 70% ──────── 90% ──── 100%
     OK (green)    Warning     Critical
                   (yellow)    (red)
```

```
  ╭────╮
  │ 27%│   ← Цвет кольца = статусный цвет
  │ CL │   ← Сокращение провайдера (CL / OA / GV)
  ╰────╯
```

```
┌──────────────────────┐
│ ● CL  27%  ● OA 72% │  ← Два провайдера в ряд
```

```
    27%
```

## 4. Garmin (Connect IQ Widget)

### 4.1 Ограничения платформы

- Разрешение: 240x240 px (типичное для Venu/Fenix)
- Цвета: ограниченная палитра (зависит от модели, обычно 64 цвета)
- Шрифт: системные Garmin-шрифты (Graphics.FONT_SMALL, FONT_MEDIUM и т.д.)

## 8. Accessibility

- Все цветовые индикаторы дублируются текстовым статусом ("OK" / "Внимание" / "Критично")
- Dynamic Type поддерживается на iOS и watchOS
- VoiceOver labels для всех интерактивных элементов
- Progress bars имеют accessibilityValue с процентом
- Контрастность цветов соответствует WCAG AA (минимум 4.5:1)
- Garmin: цвет + текстовый дубль статуса (ограничения платформы не позволяют VoiceOver)
